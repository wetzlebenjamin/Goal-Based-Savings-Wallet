# Goal-Based Savings Wallet
A smart contract for creating and managing savings goals with target amounts and deadlines.

## 🌟 Features

- Create savings goals with custom titles
- Set target amounts and deadlines
- Deposit STX towards goals
- Withdraw funds when goals are met or deadline passes
- Track progress of savings goals
- Transfer contract ownership

## 🔒 Security Features

- Input validation for amounts and deadlines
- Access control for goal management
- Safe STX transfers using contract principal
- Protected contract ownership

## 🚀 Getting Started

1. Install Clarinet
2. Clone this repository
3. Run tests: `clarinet test`
4. Deploy contract: `clarinet deploy`
